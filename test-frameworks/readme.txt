## Some popular JS projects testing
* qUnit  (used by jQuery)
* Jasmine (used by Knockout, Vue2)
* Jest (used by Vue3)

## Testing Frameworks and Libraries

* Mocha - a JS test framework
    - Provides a simple runner
    - Organize code into "suite" and "test" (or using "describe" and "it" in BDD style)
    - Provider reporter for display test results
    - For assertion library use ChaiJS.
    - For mock library use SinonJS.

* Karma - Spectacular Test Runner for Javascript
    ** Run tests on real Devices
    ** Remote control tests
    ** Integrate with other test frameworks, such as Mocha
    ** Code coverage generator

* Selenium - a browser driver library
